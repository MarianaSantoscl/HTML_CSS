Outras formatações

Código-fonte / Pré-formatação

O comando document.getElementById('teste') é escrito em linguagem JavaScript. A tag CODE faz fonte espaçada, que é ótima para a escrita de códigos, já que fica mais legível.

            
                num = int(input('Digite um número'))
                if num % 2 == 0:
                    print(f'O número{num} é PAR')
                else:
                    print(f'O número {num} é ÍMPAR)
                print('Fim do programa')

Use a tag PRE para pré formatação de um código para ficar corretamente indentado. A tag CODE deixa as letras monoespaçadas, mas não faz quebra de linhas.

Citações simples

A tag Q dá o sentido semântico de citação para o texto dela, sem fazer quebra de linha.

Como diria o pai de um amigo: o computador é um burro muito rápido.

Citações complestas

Usamos a tag BLOCKQUOTE para fazer citações completas que se destacam na página. UM adicional é o uso do parâmetro CITE, onde você pode citar a URL da informação. Isso não aparecerá na interface do site, mas a parte interna do seu site vai saber que você está referenciando aquele endereço.

Abreviações

Você pode dar um significado oculto às suas abreviações desse jeito SUS.

Ao contrário

Uma tag irrelevante é a BDO que faz o texto ficar invertido. Pode ser usado pra ficar da direita pra esquerda ou no sentido normal E-D.
